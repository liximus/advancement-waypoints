Advancement Waypoints — мод, встраивающий систему вэйпоинтов в стандартный экран достижений. Метки отображаются как обычные узлы в меню достижений, сохраняя ванильную эстетику.

### Основные возможности

Управление вэйпоинтами

- Создание вэйпоинтов через GUI с настройкой названия, описания и иконки из любого предмета
- Поддержка нескольких координат/измерений в одной метке одновременно
- Редактирование и удаление ранее созданных точек
- Привязка меток к любым родительским узлам, включая ванильные и серверные вкладки достижений

Навигация

- Компас на HUD, показывающий направление к цели
- Активация навигации кликом ЛКМ по вэйпоинту в экране достижений
- Телепортация к точке по клику ПКМ по вэйпоинту
- Автоматический выбор ближайшей координаты из нескольких записанных в метку
- Команда /navigate для быстрой навигации без создания постоянного вэйпоинта

### Совместимость

- Совместим с Better Advancements. Другие моды на изменение экрана достижений используйте на свой риск. При обнаружении ошибок можете написать мне.
